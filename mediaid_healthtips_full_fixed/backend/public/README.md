# MediaAid Health Tips Clone

A lightweight, front‑end replica of the **MediaAid Health Tips** page with modern UX enhancements.

## Features
- Responsive TailwindCSS layout
- Dark / Light mode with persistence
- Smooth card animations
- Instant Help widget with geolocation
- Accessible (ARIA labels, keyboard focusable)

## Quick Start
1. Unzip the project.
2. Open `index.html` in any modern browser — no build step required.

### Optional: Serve locally with live reload
```bash
npx serve .
# or
python -m http.server
```
